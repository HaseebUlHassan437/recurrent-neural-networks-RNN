# Recurrent Neural Networks (RNN) – Basic Implementation

This repository contains two simple Google Colab notebooks demonstrating the basics of **Recurrent Neural Networks (RNNs)** using Keras.

## 📘 Notebooks

1. **Integer Encoding with SimpleRNN**  
   Demonstrates how to preprocess text data using integer encoding and train a basic RNN model using the `SimpleRNN` layer.

2. **Sentiment Analysis with SimpleRNN**  
   A minimal sentiment analysis model trained on movie reviews using a `SimpleRNN` layer for binary classification.

## 🛠️ Tools Used
- Python
- TensorFlow / Keras
- Google Colab

## 🚀 Quick Start
You can open each notebook directly in Google Colab and run them step-by-step:
- [Integer Encoding Notebook](./integer_encoding_simplernn.ipynb)
- [Sentiment Analysis Notebook](./Copy_of_sentiment_analysis_simplernn.ipynb)
